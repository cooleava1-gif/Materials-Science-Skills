# Acta Materialia Submission Guidance

> Verify the current Guide for Authors before submission.

## Submission routes

- Research article: unstructured abstract, 250 words.
- Technical Note: verify the current word and abstract limits.
- Standard review articles are commissioned; do not use a generic review route
  without an editor invitation.

## Required package items

- Graphical abstract.
- Data availability, CRediT, competing-interest, funding, and conditional
  ethics statements.
- Keywords, figures, numbered references, and a cover letter that explains the
  processing-structure-property contribution.

Acta Materialia does not use Elsevier-style highlights in this package.

## Formatting

- Use `elsarticle.cls` with the journal's official template.
- Use numbered references in square brackets.
- Verify figure-resolution, APC, portal, and any article-type-specific limits
  against the current journal guide.
