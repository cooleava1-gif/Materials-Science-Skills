# Ceramics International Submission Guidance

> Verify the current Guide for Authors before submission.

## Submission routes

- Research Article: unstructured abstract, 200 words.
- Review Article: verify the current limits.

## Required package items

- Graphical abstract where applicable.
- Data availability, CRediT, competing-interest, funding, and conditional
  ethics statements.

Elsevier-style highlights are not a required package item for this template.

## Formatting

- Use the official Elsevier template and numbered references.
- Verify figure requirements, APC, portal, and article-type limits before
  submission.
