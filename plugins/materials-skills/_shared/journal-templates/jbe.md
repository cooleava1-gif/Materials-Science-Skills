# JBE Submission Guidance

> Always verify against the current Guide for Authors before submission.
> Official page: https://www.sciencedirect.com/journal/journal-of-building-engineering/publish/guide-for-authors

## Official class

Use Elsevier's `elsarticle.cls`. Paste this line:

```latex
\documentclass[review,numbered]{elsarticle}
```

## Metadata fill steps

1. Title, authors, affiliations, corresponding author email.
2. Structured abstract (~250 words).
3. Keywords (3-6).
4. Highlights (3-5 items, ≤85 characters each).
5. Mandatory declarations: data availability, CRediT, conflict of interest,
   funding, ethics if applicable.

## Layout notes

- Numbered references in square brackets.
- Panel labels lowercase in parentheses.
- Figure resolution and format per Elsevier standard.

## Submission checklist (assembled from journal-formats facts)

- [ ] Structured abstract within word limit
- [ ] Highlights 3-5 items, ≤85 characters each
- [ ] Keywords 3-6
- [ ] Data availability statement
- [ ] CRediT author statement
- [ ] Declaration of competing interest
- [ ] Funding statement
- [ ] Ethics statement (if applicable)
- [ ] Figures meet resolution and format requirements
- [ ] References numbered in square brackets
- [ ] Cover letter states finding, novelty, journal fit

## Live-verification placeholders

Before final submission, re-check: word limit, abstract word count,
reference cap, APC rate, submission portal URL.
