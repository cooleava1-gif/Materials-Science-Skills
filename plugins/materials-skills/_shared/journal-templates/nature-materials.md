# Nature Materials Submission Guidance

> Verify the current Nature Materials author instructions before submission.

## Submission routes

- Article: up to 3,000 main-text words.
- Letter: up to 2,000 main-text words.
- Review: 5,000-6,000 main-text words.
- Perspective: 4,000-5,000 main-text words.

Use these official publisher-facing labels, not "short communication".

## Required package items

- Data availability statement.
- Code availability statement when custom code was used.
- Author-contributions, competing-interests, and conditional ethics statements.
- No Elsevier-style highlights.

## Formatting

- Use the official Nature template and sequential numbered references.
- Verify figure requirements, APC, portal, and article-type-specific abstract
  requirements before submission.
