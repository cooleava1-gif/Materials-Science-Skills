# Functional / Electronic Materials Domain Data Guide

For functional materials data, track:

- Synthesis method, precursor purity, substrate, deposition parameters.
- Structure: XRD 2θ range, step size, reference pattern; Raman laser wavelength.
- Optical: UV-vis wavelength range, Tauc plot method (direct/indirect).
- Electrical: measurement configuration (2-probe/4-probe), temperature, atmosphere.
- Device: architecture, active area, electrode material, encapsulation.
- Electrochemical: electrolyte, reference electrode, scan rate, voltage window.
