# Metals and Alloys Domain Data Guide

For metals data, track:

- Composition (wt% of each alloying element), heat/cast number.
- Processing: melting, casting, deformation ratio, heat treatment (T, t, cooling rate).
- Microstructure: grain size (method, ASTM E112), phase fraction, precipitate size.
- Mechanical: test standard (ASTM E8/E21), specimen geometry, orientation, loading rate.
- Hardness: scale (HRC, HV, HB), load, indenter.
- Corrosion: test standard, electrolyte, temperature, exposure time, specimen surface finish.
