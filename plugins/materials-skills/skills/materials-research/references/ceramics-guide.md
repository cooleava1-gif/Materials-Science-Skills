# Ceramics Characterization and Standards Guide

Use this reference when planning or reviewing ceramics characterization and mechanical testing.

## XRD Phase Analysis

- Report: instrument, radiation (Cu Ka, Co Ka), step size, 2theta range, dwell time.
- For Rietveld refinement: report Rp, Rwp, GOF, and estimated phase fractions with uncertainty.
- For quantitative phase analysis, use internal standard or RIR method.

## SEM/EDS

- Report: accelerating voltage, working distance, detector (SE, BSE, In-lens), coating (Au/Pt/C).
- For grain size measurement: report number of grains counted, method (line intercept, image analysis), and distribution statistics.
- For EDS: report acquisition time, standard used for quantification, detection limits.

## Mechanical Testing Standards

| Property | Standard | Specimen | Key reporting parameters |
|---|---|---|---|
| Flexural strength (MOR) | ASTM C1161, ISO 14704 | 3x4x45 mm or 1.5x2x25 mm | Span, crosshead speed, number of specimens |
| Fracture toughness K_IC | ASTM C1421, ISO 18756 | SENB or SEVNB | Notch depth, pre-crack method, loading rate |
| Vickers hardness | ASTM C1327 | Polished surface | Load (kgf), dwell time, indent diagonal |
| Compressive strength | ASTM C773 | Cylinder or cube | Aspect ratio, loading rate |
| Weibull statistics | ASTM C1239 | Minimum 30 specimens | Weibull modulus, characteristic strength |

## Thermal Properties

| Property | Standard | Key reporting parameters |
|---|---|---|
| Thermal conductivity | ASTM E1461 (laser flash), ASTM C177 (guarded hot plate) | Temperature, specimen thickness |
| Thermal expansion | ASTM E228 (dilatometry) | Temperature range, heating rate |
| Specific heat | ASTM E1269 (DSC) | Heating rate, atmosphere |
| Thermal shock | Quench test (delta T at strength retention) | Quench medium, temperature difference |

## Reporting Rules

- Every mechanical property: number of specimens, mean, SD, test standard.
- Flexural strength needs Weibull analysis (min 10, ideally 30 specimens).
- Thermal conductivity reported with measurement temperature.
- Sintering conditions: atmosphere, heating rate, target temperature, dwell, cooling rate.
