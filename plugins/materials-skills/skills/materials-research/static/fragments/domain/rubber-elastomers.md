# Domain: Rubber and Elastomers

## Usage

Covers natural rubber (NR), styrene-butadiene rubber (SBR), ethylene-propylene-diene monomer
(EPDM), silicone rubber, polyurethane (PU) elastomers, and related crosslinkable elastomer
systems. Applicable to seals, gaskets, vibration damping, tires, and flexible medical devices.

## Core logic

1. Compound formulation (polymer, fillers, plasticizers, curatives) determines processability and base properties.
2. Vulcanization system (sulfur, peroxide, radiation) and cure schedule set crosslink type and density.
3. Crosslink structure (mono-, di-, polysulfidic; network homogeneity) governs elastic recovery, heat aging, and fatigue.
4. Mechanical and aging performance (tensile, elongation, compression set, ozone resistance) are validated at application conditions.
5. Service environment (temperature, media, dynamic strain) must match testing conditions for credible lifetime claims.

## Key evidence categories

- Cure characteristics: rheometer (tc90, minimum/maximum torque, scorch time)
- Mechanical: tensile (strength, modulus at 100%/300%), elongation at break, Shore hardness, compression set
- Dynamic: DMA (storage/loss modulus, tan delta), dynamic fatigue
- Thermal: DSC (Tg, crystallization), TGA (thermal stability)
- Aging: heat aging (70/100/125°C), ozone exposure, fluid immersion (oil, fuel, acid)
- Morphology: SEM (filler dispersion, fracture), TEM (nanofiller distribution)

## Reviewer risks

- Vulcanization parameters (temperature, time, pressure) not reported; cure state assumed rather than verified.
- Dynamic mechanical data absent; static tensile data alone cannot predict vibration or fatigue performance.
- Overclaiming from lab specimen (dumbbell) to product-scale component without scale-up validation.
- Compression set tested at single temperature only; no Arrhenius extrapolation for long-term sealing claims.
- Filler type and loading (carbon black grade, silica surface area) insufficiently specified for reproducibility.
