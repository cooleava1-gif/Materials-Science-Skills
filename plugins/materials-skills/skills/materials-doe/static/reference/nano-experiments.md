# Nanomaterials Experiment Design Reference

## Typical factors
- Precursor: type, concentration, ratio.
- Synthesis: temperature, time, pH, stirring rate.
- Capping agent: type, concentration, addition method.
- Post-treatment: annealing T/t, purification method, surface modification.

## Typical responses
- Size: mean diameter (TEM), hydrodynamic size (DLS), crystallite size (XRD).
- Morphology: aspect ratio, shape distribution, uniformity.
- Optical: absorption peak, PL intensity, bandgap.
- Colloidal: zeta potential, PDI, aggregation over time.
- Catalytic: conversion rate, selectivity, turnover frequency.

## Recommended design
- Full or fractional factorial for synthesis parameter screening.
- RSM for size/property optimization.
- Custom design for multi-step synthesis optimization.
