# Polymers Experiment Design Reference

## Typical factors
- Polymer type, molecular weight, grade.
- Modifier/filler type and loading (phr, wt%).
- Processing temperature, pressure, cooling rate.
- Post-processing: annealing T/t, crosslinking agent ratio.

## Typical responses
- Mechanical: tensile strength, elongation, modulus, hardness.
- Thermal: Tg, Tm, Td, thermal conductivity.
- Rheological: viscosity, melt flow index, gel time.
- Morphological: dispersion quality, phase size.

## Recommended design
- Full factorial for ≤4 factors at 2–3 levels.
- Central composite design (CCD) for process optimization.
- Mixture design for blend/composite formulation.
