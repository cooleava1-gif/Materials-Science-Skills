# 02 Evidence Table

Use this table before drafting or revising prose. Metadata, memory, and
abstract-only screening can guide priority, but manuscript claims need source
text, data, figures, tables, or an explicit user-provided record.

## Status Vocabulary
- `evidence-backed`: direct data, figure, table, source text, or verified record.
- `plausible-inference`: reasonable interpretation with indirect support.
- `hypothesis`: possible explanation that needs new evidence.
- `unsupported`: not usable as a manuscript claim.
