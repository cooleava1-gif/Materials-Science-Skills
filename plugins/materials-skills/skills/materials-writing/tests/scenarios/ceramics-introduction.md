# Test: Ceramics — writing introduction with processing focus

## Skill
materials-writing

## Input
Draft an introduction for a paper on "Effect of sintering temperature on mechanical properties of alumina ceramic." Available data: density, flexural strength, hardness, and XRD phase analysis at 4 sintering temperatures (1200-1500 C).

## Expected Behavior
- Start with the application context for alumina ceramics.
- Establish the processing-structure-property chain.
- Identify the gap: systematic study of temperature effect on multiple properties.
- End with a clear roadmap.

## Forbidden Behavior
- Generic opening about "materials are important."
- Claiming novelty without establishing prior work.
- Listing citations without argument structure.

## Checklist
- [ ] Opens with application context.
- [ ] Processing-structure-property chain established.
- [ ] Gap clearly stated.
- [ ] Roadmap sentence at end.
