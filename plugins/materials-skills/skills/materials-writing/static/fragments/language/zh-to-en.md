# Language: Chinese-to-English (writing)

Use this when the user's notes are Chinese, mixed Chinese-English, or organized as lab notes rather than manuscript prose.

## Translate intent, not syntax

Chinese academic notes often pack background, motivation, method, and implication into one long sentence. Before drafting English, split each note into:

- claim
- evidence
- condition
- comparison
- implication
- limitation

Then write English in the order required by the section, not in the order of the Chinese sentence.

## Common repairs

| Chinese-draft pattern | English repair |
|---|---|
| Broad importance before a clear object | Name the system or problem earlier |
| Method list before research gap | Move the gap before the method |
| `显著提高 / 明显改善` without baseline | Add the comparator or soften the verb (`increased`, not `significantly improved`) |
| `首次 / 创新性` without scope | Replace with a bounded novelty claim |
| Mechanism inferred from correlation | Use `suggests`, `is consistent with`, or ask for mechanistic evidence |
| Results mixed with implications | Put observation in Results, meaning in Discussion |
| Repeated topic noun where English would use a pronoun | Replace or elide |
| Strings of short clauses joined by commas | Split or add explicit connectives |
| `本文` opening (`本文研究了…`) | Lead with the finding, not with `In this paper we…` |
| Topic-comment structure (`…的研究，表明…`) | Reorder to subject-verb-object |
| Stacked attributives (`…的…的…的`) | Break into two sentences or use a relative clause |
| Tense drift (mixed past/present) | Past tense for what was done; present for general facts |

## Materials terminology mapping (examples)

Use these as defaults; confirm against the domain terminology ledger.

| Chinese term | English default | Notes |
|---|---|---|
| 水化 | hydration | `hydration products`, not `hydration effect` |
| 掺量 | dosage / content | `dosage` for additives; `content` for composition |
| 配合比 | mix proportion | not `mix ratio` in concrete context |
| 改性 | modification | `modified asphalt`, not `modified asphalt material` |
| 力学性能 | mechanical properties | not `mechanical performances` |
| 耐久性 | durability | specify the mechanism (freeze-thaw, chloride, etc.) |
| 粘结 / 粘结性能 | bonding / bonding performance | `interlayer bond strength` for pavement context |
| 抗水损害能力 | moisture resistance | or `moisture-conditioned retention` if data supports |
| 微观形貌 | microstructure / morphology | `morphology` for SEM images; `microstructure` for the broader concept |
| 机理分析 | mechanism discussion | only if evidence is supplied |
| 显著提高 | increased / improved | `significant` only with statistics |

## Output convention

For Chinese-author notes, provide polished English **first**, then brief Chinese notes explaining major structural choices (tense, claim strength, split points). This lets a Chinese author verify intent without re-reading the English from scratch.

## Keep claim strength below the evidence ceiling

Do not promote a `consistent with` finding to `demonstrates` wording. If the Chinese note claims `证明了`, check whether the evidence actually supports `demonstrate` or only `suggests`; downgrade if needed and flag the change in the Chinese notes.
