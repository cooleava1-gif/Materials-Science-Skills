# Writing Mode: hybrid

Use `hybrid` when the user has a partial draft and also needs expansion,
restructure, or new sections.

## Required Moves
- Reconcile existing text with `01_research_canon.md` and `02_evidence_table.md`.
- Mark old draft material as keep, revise, move, or defer.
- Build missing sections from `04_section_contracts.md`, not from style alone.
- Track unresolved structural decisions in `state.json.technical_debts`.

## Output Bias
- Separate retained evidence-backed text from newly drafted placeholders.
- Stop if old claims conflict with the canon or if expansion needs new evidence.
