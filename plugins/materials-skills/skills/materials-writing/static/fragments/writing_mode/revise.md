# Writing Mode: revise

Use `revise` when the user provides an existing paragraph, section, manuscript,
proposal, or reviewer-driven draft and wants improvement.

## Required Moves
- Backfill foundation files from the existing draft before editing.
- Identify which claims are already supported, inferred, or unsupported.
- Revise only the targeted paragraphs unless the foundation reveals a structural conflict.
- Update `state.json` scores and technical debts after the revision.

## Output Bias
- Preserve correct text verbatim.
- Prefer a revision brief when missing evidence prevents a safe rewrite.
