# Writing Mode: compose

Use `compose` when the user has a topic, direction, claims, or notes but no
stable draft. Initialize or request the foundation pack before prose.

## Required Moves
- Fill `00_scope.md` enough to name the task, reader, deliverable, and blocked claims.
- Lock facts and terminology in `01_research_canon.md`.
- Sketch `02_evidence_table.md` and `03_argument_map.md`.
- Draft only after the one-sentence argument and section contract are coherent.

## Output Bias
- Prefer foundation gaps and targeted questions over fluent but unsupported prose.
- If evidence is missing, stop at foundation readiness and report the next action.
