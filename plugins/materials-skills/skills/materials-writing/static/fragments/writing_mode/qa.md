# Writing Mode: qa

Use `qa` when the user asks to audit, score, check, or decide whether a draft
is ready for the next writing step.

## Required Moves
- Run Gate 2 content review before language cleanup.
- Score the eight rubric dimensions and route by the lowest dimension.
- Apply stopping rules before recommending another full revision loop.
- Produce a QA report or revision brief without rewriting prose unless requested.

## Output Bias
- State pass, revise, or stop.
- Include current score/status, remaining risks, stop-or-continue reason, and next action.
