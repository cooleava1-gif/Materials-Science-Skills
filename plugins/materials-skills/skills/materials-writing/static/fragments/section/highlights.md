# Highlights

Use this fragment when the user asks for journal highlights or a bullet summary.

## Rules

- 3–5 bullets
- Each bullet ≤ 85 characters including spaces (common journal limit)
- Focus on what is new, not what was done
- Avoid abbreviations unless defined
- Begin with a noun or verb, not a connecting word

## Template

```text
- [Novel material/system] shows [key performance] under [condition].
- [Mechanism] explains the observed [improvement/trend].
- [Method/design] enables [benefit] for [application].
- Results are supported by [characterization evidence].
```
