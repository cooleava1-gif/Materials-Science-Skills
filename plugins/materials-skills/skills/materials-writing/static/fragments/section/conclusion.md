# Conclusion

> **Domain context**: The `domain` axis has loaded domain-specific writing guidance for [detected domain]. The conclusion must reflect the domain's evidence standards — avoid overclaiming in areas where the field is sensitive.

Conclusions should summarize verified contributions, not introduce new claims. Use this four-part structure:

1. **Contribution** — the main performance or design finding, stated plainly.
2. **Limitation** — what the evidence does not cover.
3. **Next step** — the most important follow-up experiment, validation, or analysis.
4. **overclaim boundary** — a sentence that blocks universal claims from bounded data.

## Overclaim blockers

Replace these with bounded language:

- "excellent" → state the measured value and the test condition.
- "significant" → only if supported by statistics; otherwise use "observed" or "measured".
- "novel" / "first" → replace with the precise gap that was closed.
- "field-ready" / "engineering application" → only if field or simulated-service data exist.
- "proves" / "confirms" → downgrade to "supports", "is consistent with", or "indicates" for single-technique evidence.

## Domain-specific boundary notes

- **Civil materials**: specify lab vs. field conditions; avoid claiming field performance without field trial data.
- **Ceramics**: note specimen size effect and surface quality influence on measured properties.
- **Metals**: acknowledge anisotropy, weldability concerns, or scaling from lab to production.
- **Polymers**: note aging, recyclability, and processing window limitations.
- **Functional**: specify device operating conditions and stability test duration.
- **Nanomaterials**: note batch-to-batch reproducibility and colloidal stability.

## Domain checklist

- [ ] Does the conclusion avoid introducing new data or citations?
- [ ] Is each quantitative claim traceable to a specific result section?
- [ ] Are limitations stated explicitly (not hidden in "future work")?
- [ ] Does the wording match the evidence strength (avoid "proves", "confirms" for single-technique evidence)?
- [ ] Does the final sentence bound the contribution rather than inflate it?
