# Stopping Rules

Stop a revision loop when continuing would hide risk, waste review effort, or
imply evidence that does not exist.

## Stop Conditions

1. Maximum three full revision rounds.
2. Two consecutive score improvements below 0.5.
3. Missing key evidence blocks a manuscript-level claim.
4. An unresolved specialist or project-constraint conflict remains.
5. The declared target threshold has been reached.

When stopping, report the current score/status, remaining risks, why the loop
stopped, and one next action. Continue only when the next revision has a clear
target and the required evidence exists or is explicitly marked missing.
