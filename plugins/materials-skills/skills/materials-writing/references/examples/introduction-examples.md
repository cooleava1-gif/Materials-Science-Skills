# Introduction Examples Index

Materials introductions narrow along "engineering bottleneck → design rationale
→ validation strategy". Pick the variant whose gap shape matches your story.

1. Engineering bottleneck — `introduction/variant-1-engineering-bottleneck.md`
   `Use when the gap is a performance, mechanism, or durability limit in an applied material.`
2. Trade-off contradiction — `introduction/variant-2-tradeoff-contradiction.md`
   `Use when existing strategies each sacrifice a different property and the contradiction is unresolved.`
3. Mechanism debate — `introduction/variant-3-mechanism-debate.md`
   `Use when performance is known but the mechanism is contested and a key characterization is missing.`
4. Performance ceiling — `introduction/variant-4-performance-ceiling.md`
   `Use when a hard performance ceiling exists and a new design route is the entry point.`
