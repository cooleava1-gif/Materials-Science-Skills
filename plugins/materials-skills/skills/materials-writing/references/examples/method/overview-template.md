# Method Overview Template

`Overview usually includes setting, core contribution, optional figure pointer, and subsection map. Distilled from the nature-writing overview-template style, adapted for materials.`

```latex
% Overview
% One or two sentences for setting — name the material system and the question
%% Example 1: [材料体系] was prepared by [制备路线] to test whether [设计思路] can improve [性能指标] under [服役条件].
%% Example 2: To clarify the [机制争议] in [材料家族], [材料变体] was synthesized across [变量范围] and characterized by [表征集合].

% One or two sentences for core contribution — name the design and the validation
%% Example 1: We designed [材料变体] with [结构特征] and evaluated [性能集合] under [标准号], combined with [表征方法] to link [结构] to [性能].
%% Example 2: Building on [前期工作/策略], we introduced [改性] at [配比] and measured [性能指标] alongside [表征] to test [假设].

% If the synthesis/test matrix is novel, point to figure
%% Example: The experimental program is illustrated in Figure [x].

% Subsection map — what each subsection covers
%% Example 1: Section [x.1] describes the synthesis and specimen preparation.
%% Example 2: Section [x.2] details the characterization techniques and their purposes.
%% Example 3: Section [x.3] reports the performance testing standards and conditions.
```

## Usage

- The setting sentence must name the material system AND the question — not just "we studied [材料]".


- The core contribution sentence must name the design AND the validation — a design without a validation route is incomplete.
- The subsection map should let a reader navigate to the element they care about (synthesis / characterization / performance).
