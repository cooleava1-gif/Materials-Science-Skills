# Introduction Variant 3 — Mechanism Debate

`Use when performance is known but the mechanism is contested and a key characterization is missing. Distilled from introduction-gap-ladder.md Move 3 and published-article-patterns Pattern J.`

```latex
% Move 1: performance improvement is already known
[性能改善] in [材料家族] through [已知策略] has been widely reported, with [性能指标] increasing by [变化幅度] under [测试条件] ([文献范围]).

% Move 2: the mechanism is contested — name the competing explanations
The origin of this improvement, however, remains debated: [派别A] attribute it to [机制A] based on [证据类型A], while [派别B] argue for [机制B] based on [证据类型B] ([文献范围]).

% Move 3: why the debate persists — the missing discriminating evidence
The debate persists because previous studies rarely report [关键表征/测试条件] together with [性能数据], so [机制A] and [机制B] cannot be distinguished for [材料体系].

% Move 4: this study's characterization strategy
Here, we combine [表征技术A] (to probe [结构特征A]) with [表征技术B] (to rule out [替代解释]) across [变量范围], alongside [性能测试], to discriminate between [机制A] and [机制B].

% Move 5: the finding the paper will defend
We show that [可观察特征] is consistent with [机制A] rather than [机制B] within [测试范围], which reframes [性能调控] as a [结构/工艺参数] problem.
```

## Usage

- Move 2 must name BOTH competing mechanisms and the evidence each side cites.
- Move 3 must identify the specific discriminating evidence that is missing — not just "more studies are needed".
- Move 4 must explain why combining [表征技术A] + [表征技术B] can discriminate where prior work could not.

## Gap quality check

The debate framing is weak if Move 5 merely confirms one mechanism. The entry must show how the combined characterization excludes the alternative, not just supports the preferred one.
