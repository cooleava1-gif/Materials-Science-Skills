# Materials Writing Example Library Index

Reusable skeletons and sentence logic distilled from `published-article-patterns.md`
and the section-pattern factories. All templates use placeholders — no real paper
titles, authors, DOIs, or data are cited.

## Usage

1. Pick a template that matches your claim shape.
2. Open the matching example file below.
3. Reuse the sentence logic, not the exact wording.
4. Keep claim-evidence-boundary anchoring at every move.
