# Section: Materials and Methods

Methods should be specific, complete, transparent, and reproducible.

Another group should be able to determine:

- what materials were used (source, purity, grade)
- what composition or mixture design was employed (ratios, dosages, w/b ratio)
- what processing conditions were applied (temperature, time, mixing speed, curing regime)
- what specimen geometry and dimensions were tested
- which test standards were followed (ASTM, ISO, GB)
- what equipment was used (model, manufacturer, key settings)
- how many replicates were tested and what statistical methods were applied

It is acceptable to abbreviate by citing an earlier report only when that report truly contains the necessary detail.

## Forbidden vague phrases in materials Methods

Never leave wording such as:

- `under standard conditions` → specify temperature, humidity, duration
- `using routine methods` → cite the specific protocol or describe the procedure
- `mixed thoroughly` → specify mixing speed (rpm), duration (min), sequence (low-speed then high-speed)
- `heated at high temperature` → specify exact temperature (°C), heating rate (°C/min), holding time
- `added in appropriate amount` → specify dosage (wt%, vol%, or absolute mass)
- `data were analyzed statistically` → specify test (ANOVA, t-test), software, significance level
- `samples were cured` → specify curing regime (temperature, humidity, duration, method)
- `the method was validated` → specify validation criteria and results

## Materials-science-specific requirements

### Materials
- List all raw materials with supplier, grade, and key properties (e.g., "Portland cement (P.O 42.5, Conch Cement, China); specific gravity 3.15, Blaine fineness 350 m²/kg").
- For nanomaterials: specify morphology, functionalization, dimensions (e.g., "carboxyl-functionalized GO, 1-5 μm lateral size, 0.8-1.2 nm thickness, 10-20% oxygen content").

### Mixture design
- Provide complete mix proportions in a table: cement, water, admixtures, dosages, w/b ratio.
- If multiple mixtures, label them clearly (M0, M1, M2...) and explain the variable being tested.

### Specimen preparation
- Specify molding method (casting, vibration, extrusion).
- Specify specimen dimensions (e.g., 50 mm cubes, 40×40×160 mm prisms).
- Specify demolding time and curing regime (e.g., "demolded after 24 h, cured in fog room at 20±2 °C and >95% RH until testing age").

### Testing
- Cite test standards explicitly (e.g., "Compressive strength was tested per ASTM C39 / GB/T 50081").
- Specify loading rate (e.g., "0.5 MPa/s"), equipment (e.g., "MTS 810 universal testing machine, 300 kN capacity").
- Specify number of replicates (e.g., "five specimens per mixture per testing age").

### Characterization
- For SEM: specify sample preparation (gold sputtering, vacuum drying), accelerating voltage, magnification range.
- For XRD: specify scan range (2θ), step size, scan speed, radiation type (Cu Kα).
- For TGA/DSC: specify heating rate, atmosphere (N₂, air), temperature range.

## Polishing priorities

- Replace all forbidden vague phrases with specific parameters.
- Ensure chronological order: materials → mixing → molding → curing → testing.
- Verify that all materials mentioned in Results are described in Methods.
- Check that test standards are current (not superseded versions).
- Ensure units are consistent (SI preferred; if using non-SI, define conversion).
