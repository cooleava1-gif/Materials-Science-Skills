# Corpus-Derived Patterns

Patterns extracted from paired paper-patent examples in the civil materials
domain. Use them as priors when drafting; do not copy verbatim.

## Problem phrasing (background)

| Pattern | Example |
|---|---|
| "现有的 X 存在 Y 的不足" | 现有的普通混凝土存在强度不足的不足 |
| "传统方法在 Z 方面存在短板" | 传统养护方法在能耗方面存在短板 |
| "亟需一种..." | 亟需一种兼顾高强度与低能耗的制备方法 |

## Solution phrasing (summary)

| Pattern | Example |
|---|---|
| "针对 X 的 Y 问题，提供一种 Z，其特征在于..." | 针对混凝土早期强度低的问题，提供一种早强型混凝土，其特征在于... |
| "包括 A、B、C" | 包括水泥、骨料和外加剂 |
| "其中 A 为 ... 选自 ... 之一" | 其中外加剂为减水剂或早强剂之一 |

## Method step phrasing

| Stage | Verbs |
|---|---|
| 备料 | 提供、称取、预处理 |
| 混合 | 混合、搅拌、分散 |
| 成型 | 浇注、振捣、压制、挤出、注模 |
| 养护 | 养护、标准养护、自然养护、蒸压、碳化 |
| 测试 | 测试、表征、检测、测量 |

## Composition phrasing

- Use ranges with units: "30-50 wt%"
- Use "选自 ... 之一" for alternatives
- Use "优选" for preferred embodiment
- Use "包括但不限" when the embodiment does not exhaust the scope

## Beneficial effect phrasing

Avoid vague claims:

- 抗压强度提高 30% 以上
- 28 天抗压强度达到 50 MPa
- 孔径分布 10-50 nm

Avoid:

- 性能优异
- 显著提高
- 大大改善
- 取得突破性进展

## Range honesty

A range claim needs at least two data points spanning the range in the
source. A "compressive strength of 50-80 MPa" without data points in that
span is unsupported. Disclose the actual measurement if you cannot span
the range.
