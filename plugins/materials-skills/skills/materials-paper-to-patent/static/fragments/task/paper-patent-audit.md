# Task: Paper-Patent Audit

Compare a supplied paper with an existing patent. Produce:

- paper-only, patent-only, and overlap feature tables with exact locators;
- bidirectional relationship labels (paper-supported, paper-broader,
  patent-only, narrower, identical, divergent);
- a `corpus-pair-audit.json` report using `references/corpus-pair-audit.md`
  schema.

No claim drafting. The output supports an infringement or
patentability discussion by a patent professional.
