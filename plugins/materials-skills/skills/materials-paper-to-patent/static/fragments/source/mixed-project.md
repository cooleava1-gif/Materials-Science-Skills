# Source: Mixed Project

Use when the workspace contains the manuscript plus supplementary files,
source code, raw data CSVs, inventor notes, or an existing patent
application.

## Inventory

List every file with:

- path;
- file type;
- what evidence it adds (e.g., "raw XRD pattern", "implementation code",
  "prior patent claim 1");
- candidate stable ID prefix (`P` for prose, `E` for equations, `F` for
  figures, `C` for code/supplementary).

## Risk flags

- Source code may contain implementation details the manuscript omits.
  Decide case-by-case whether to include in claims.
- Raw data CSVs may show trends the manuscript paper omits — useful for
  embodiments, risky for claims.
- An existing patent defines a prior-art boundary — read it before drafting
  independent claims.
