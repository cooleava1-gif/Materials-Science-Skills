# Durability Figures

> **Domain context**: The `domain` axis has loaded domain-specific figure guidance. Durability figures vary significantly by domain — the domain guide lists field-specific aging/stress conditions.

Use retention ratios and condition labels.

Examples:

- **Civil**: dry versus wet bonding, before/after ageing, freeze-thaw cycles, chloride/carbonation/sulfate exposure.
- **Polymers**: UV exposure, thermal ageing (accelerated), hydrolytic degradation, creep over time.
- **Metals**: corrosion (salt spray, immersion, cyclic), fatigue life (S-N curves), creep rupture.
- **Ceramics**: thermal shock (ΔT critical), thermal cycling, steam/humidity exposure.
- **Functional**: performance retention vs time/cycles, thermal stability of efficiency.
- **Nanomaterials**: colloidal stability over time, agglomeration under relevant media conditions.

Durability captions must state the stress condition (temperature, duration, cycles, medium), not only the percentage change.
