# Polymers Figure Package

Recommended package for polymer and composite characterization:

1. Processing–structure–property overview (formulation or processing route vs key property),
2. Thermal analysis: DSC traces or TGA/DTG curves showing Tg, Tm, crystallinity, or degradation,
3. Mechanical property comparison: tensile strength/modulus or impact strength across formulations,
4. Stress–strain curves for representative samples showing ductile–brittle transition,
5. Morphology plate: SEM fracture surfaces or TEM dispersion images,
6. Durability retention: property retention after aging (thermal, UV, hydrolysis),
7. FTIR or Raman evidence for chemical changes (curing, degradation, filler–matrix interaction).

Caption stance: structure–property link and processing trade-offs are more important than "higher filler = better". Report dispersion quality alongside property gains for composites.
