# Functional Materials Figure Package

Recommended package for functional materials characterization:

1. Composition–property map (dopant concentration vs dielectric/piezoelectric/transport property),
2. Dielectric permittivity and loss tangent vs frequency and/or temperature,
3. Ferroelectric P-E hysteresis loops for different compositions or conditions,
4. Impedance spectroscopy (Nyquist plot) separating grain and grain-boundary contributions,
5. Optical band gap determination (Tauc plot from UV-Vis),
6. Temperature-dependent property evolution (Curie temperature, depolarization, phase transition),
7. Device performance: efficiency, sensitivity, or response time vs operating condition.

Caption stance: measurement conditions (frequency, temperature, field) must accompany every property value. Distinguish intrinsic material properties from extrinsic device performance.
