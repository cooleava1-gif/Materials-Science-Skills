# Archived Reference Script

`plot_ceramics_atlas.py` is an **archived reference example** for generating
the ceramics characterization atlas.

**Status**: This script depends on `materials_plot_lib` which was removed when
the skill transitioned to "LLM-as-artist" mode. The ceramics-atlas assets
have also been removed. It is kept as a reference for the ceramics atlas
figure patterns.

**How to use**: Read this script for ceramics characterization figure patterns
(XRD, stress-strain, TGA/DSC, Weibull, etc.). Do not run it directly — instead,
ask the LLM to generate new plotting code based on the patterns shown here,
or refer to `assets/materials4papers/` for runnable Nature-style examples.
