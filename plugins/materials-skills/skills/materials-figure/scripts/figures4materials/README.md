# Archived Reference Scripts

These 65 plotting scripts are **archived reference examples** from the
materials-figure skill. They demonstrate publication-ready matplotlib patterns
for materials characterization figures.

**Status**: These scripts are reference-only. They depend on `materials_plot_lib`
which was removed when the skill transitioned to "LLM-as-artist" mode. The LLM
now writes plotting code directly in `plot.py` for each figure package.

**How to use**: Read these scripts for chart-type patterns, style conventions,
and data organization. Do not run them directly — instead, ask the LLM to
generate a new `plot.py` based on the pattern you need.

**Data files**: The `data/` subdirectory contains CSV source data files that
are still usable as templates for new figure packages.
