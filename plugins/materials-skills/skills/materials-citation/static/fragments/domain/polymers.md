# Polymers Domain Citation Rules

For polymer and composite manuscripts, keep evidence separated by these layers:

- **Synthesis/Formulation**: monomer, initiator, curing agent, stoichiometric ratio, molecular weight (Mw, Mn, PDI).
- **Processing**: method (injection molding, extrusion, compression molding, hand lay-up), full parameter set (temperature, pressure, speed, cooling rate, curing schedule).
- **Filler/Interface**: filler type, geometry (fiber, particle, platelet), content, surface treatment, dispersion method.
- **Structure-Property**: crystallinity, crosslink density, morphology (SEM/TEM/AFM), chain architecture.
- **Thermal/DSC**: Tg, Tm, crystallinity, degradation onset, char yield, DMA (storage/loss modulus, tan delta).
- **Durability**: UV aging, thermal aging, hydrolysis, oxidation induction time, recyclability cycling.

Separate processing-condition citations from mechanism citations. Filler dispersion evidence (SEM/TEM) is required before attributing property gains to nanoscale reinforcement.
