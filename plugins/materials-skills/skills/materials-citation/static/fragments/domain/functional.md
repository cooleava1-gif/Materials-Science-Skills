# Functional Domain Citation Rules

For functional materials manuscripts, keep evidence separated by these layers:

- **Composition/Doping**: base material, dopant type and concentration, defect chemistry, solid solution range.
- **Synthesis/Processing**: powder synthesis, forming (pressing, tape casting, printing), sintering/deposition conditions (temperature, atmosphere, time).
- **Phase/Structure**: crystal structure (XRD), phase purity, grain size, domain structure, defect concentration.
- **Electrical properties**: conductivity (ionic/electronic), dielectric constant, loss tangent, piezoelectric coefficient (d₃₃, d₃₁), ferroelectric polarization.
- **Optical properties**: band gap, refractive index, photoluminescence, transmittance, nonlinear optical coefficients.
- **Thermal properties**: thermal conductivity, thermal expansion, specific heat, thermal stability.
- **Device performance**: efficiency, sensitivity, response time, cycling stability, operating temperature range.

Separate material-property citations from device-performance citations. Measurement frequency and temperature must be stated before comparing dielectric or impedance data across studies.
