# Journal Search Profiles

> **Domain context**: The `domain` axis has loaded domain-specific citation guidance. Supplement these journal search profiles with the domain guide's search strategy recommendations.

## CBM and JBE (Construction and Building Materials)

Use broad construction-materials queries first, then narrow by material system and test method.

Useful concept blocks:

- construction material performance,
- modified asphalt or cement/concrete system,
- durability and service environment,
- microstructure or mechanism evidence,
- application or engineering feasibility.

## CCC (Cement and Concrete Composites)

Search for composite-material mechanisms, cementitious matrix interactions, hydration, microstructure, interfacial transition zones, and durability mechanisms.

CCC-style support should not rely only on applied strength data.

## RMPD and IJPE (Pavement Engineering)

Search pavement-material performance, bituminous binder/emulsion behavior, mixture properties, traffic loading, moisture damage, aging, field relevance, and standard test conditions.

RMPD/IJPE citation maps should separate binder-level, mixture-level, and pavement-service evidence.

## Ceramics journals (JACerS, Ceramics International, JEurCerSoc)

Search powder synthesis, forming methods, sintering optimization, phase evolution, mechanical properties (strength, toughness, Weibull), thermal/functional properties.

Separate processing citations from property citations. Report sintering conditions precisely.

## Metals journals (Acta Materialia, Scripta Materialia, Metallurgical Transactions)

Search composition design, thermomechanical processing, microstructure characterization, mechanical properties, corrosion/environmental performance.

Separate processing-microstructure citations from property-environmental citations.

## Polymers journals (Polymer, Composites, Macromolecules)

Search polymer synthesis, thermal characterization, mechanical/rheological properties, composite interface, aging/degradation.

Separate synthesis-structure citations from property-application citations. Report molecular weight characterization.

## Functional materials journals (Advanced Materials, ACS Nano, Nature Electronics)

Search synthesis/deposition, structural characterization (XRD, Raman, XPS), functional properties, device performance, stability.

Separate synthesis-structure citations from device-performance citations. Include benchmark comparisons.

## Nanomaterials journals (ACS Nano, Nano Letters, Nano Today)

Search synthesis, size/morphology control, surface chemistry, size-dependent properties, application demonstration.

Separate synthesis-characterization citations from property-application citations. Require size statistics.
