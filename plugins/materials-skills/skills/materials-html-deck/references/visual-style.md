# Visual Style

Use a high-fidelity academic HTML deck style:

- fixed 1920 x 1080 browser canvas,
- action title or assertion title on every evidence slide,
- short titles when the slide is diagram-led,
- high contrast,
- figures larger than explanatory text,
- source notes and captions visible but quiet,
- one main claim per slide,
- visual rhythm across cover, evidence, metric, diagram, and summary slides.

Recommended material color roles:

- asphalt/pavement: graphite plus amber or teal accent,
- cement/concrete: warm gray plus blue accent,
- sustainability: muted green plus graphite,
- polymer/resin: deep ink plus controlled cyan or green accent.

Avoid single-template monotony. Rotate visual roles: full-title cover, single
evidence figure, metric cards, diagrammatic mechanism, bento synthesis, and
closing takeaway.
