# Task: HTML Deck

Produce a browser-native HTML academic deck. The deck directory is the delivery
artifact and must include the slide HTML, index shell, screenshots, QA report,
speaker notes, and asset manifest.

Use `scripts/build_deck.mjs` for generation and strict verification. If the
outline names an `academic_style`, honor it unless the CLI `--style` overrides
it.

Do not add export artifacts to the default task.
