# Reader Note — Domain Context

Loaded automatically when the domain is nanomaterials.

When reading a nanomaterials paper, pay special attention to:
- Synthesis method and parameters (precursor, capping agent, T/t).
- Size characterization: TEM with statistics (≥100 particles) and DLS with PDI.
- Structural confirmation: XRD, Raman, UV-vis.
- Surface chemistry: functionalization, zeta potential.
- Colloidal stability assessment.

Record the synthesis method, size statistics, structural data, and property-performance relationships.
