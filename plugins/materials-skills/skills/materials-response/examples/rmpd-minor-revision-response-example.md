# RMPD Minor Revision Response Example

## Reviewer Comment

The experimental section states that the Thin-Film Oven Test (TFOT) was conducted at 160 °C. Standard specifications (ASTM D1754 / AASHTO T 179) dictate a testing temperature of 163 °C. Please clarify if this was a deviation from the standard or a typo, and verify the aging index results. Additionally, the discussion on the rutting resistance of the polymer-modified emulsified asphalt would be strengthened by citing recent literature (2024-2025) on Multiple Stress Creep Recovery (MSCR) parameters ($J_{nr}$ and $R$).

## Good Response

We thank the reviewer for pointing out these minor errors and for recommending key references. The response should verify whether the temperature mismatch is a typo or a real protocol deviation before drafting the final answer.

**Concern:** Temperature mismatch with standard ASTM D1754 / AASHTO T 179 specifications.
**Response:**

If it was a typo, state the verified standard condition and correct the text. If it was a deviation, explain the reason, recalculate or qualify the aging index results, and add the limitation. `[AUTHOR_INPUT_NEEDED: verify TFOT temperature from lab records and confirm whether Table/Figure values remain valid.]`

**Manuscript Action:**
1. Corrected or qualified the TFOT temperature on Page [P], Section [X], Lines [L-L].

```diff

+The binder was aged using the Thin-Film Oven Test (TFOT) at [verified temperature] for [verified duration] per [verified standard], or the deviation is now explained in Section [X].
```

## Why This Works

This response is effective because:
1. It handles minor issues swiftly and politely without unnecessary defensiveness.
2. It requires factual verification of the temperature discrepancy before claiming a typo or a protocol deviation.
3. It corrects the typo with a clear, standard-compliant edit, referencing the exact specification (ASTM D1754).
4. It accepts the suggestion to cite recent works while preventing invented reference names, values, or literature claims.
5. It points precisely to page and line numbers in the revised draft.
