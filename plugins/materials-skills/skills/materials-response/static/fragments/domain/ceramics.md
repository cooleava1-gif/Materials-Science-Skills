# Reviewer Response — Domain Context: Ceramics

When responding to ceramics reviewer comments:
- Density/porosity without sintering details is commonly questioned.
- Weibull statistics with insufficient specimens is a frequent concern.
- Toughness measurement method (KIC) is often challenged.
- Phase identification without reference matching is flagged.
- Sintering profile (heating rate, dwell, atmosphere) without full details is raised.

Provide complete processing parameters and statistical evidence for each response.
