# Submission Checklist Template

This checklist is assembled by `generate_checklist.py` from the
journal-formats fact sheet and the journal-templates yaml. Do not fill it
by hand; run the script.

## Generic items (always present)

- [ ] Title
- [ ] Authors, affiliations, corresponding author
- [ ] Abstract within word limit
- [ ] Keywords
- [ ] Declarations (data availability, CRediT, conflict of interest, funding, ethics if applicable)
- [ ] Figures meet resolution and format requirements
- [ ] References in journal style
- [ ] Cover letter states finding, novelty, journal fit
