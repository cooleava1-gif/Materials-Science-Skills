# Highway Engineering Reviewer Criteria

## Top 10 Reviewer Concerns for Highway / Pavement Papers

### 1. Traffic Data Not Specified
- Traffic level (ESALs) must be reported for all sections
- Growth rate and design period must be stated
- Do not compare sections with vastly different traffic levels

### 2. Climate Zone Ignored
- Temperature range, freeze-thaw cycles, and rainfall must be reported
- Pavement performance is climate-dependent; results may not transfer
- Specify PG binder grade or equivalent climate classification

### 3. Insufficient Monitoring Period
- Minimum one full seasonal cycle for rutting and roughness
- Cracking may need 3–5 years to develop
- Do not extrapolate short-term data to long-term performance

### 4. Lab-to-Field Gap
- Lab rutting tests (e.g., Hamburg, APA) do not directly predict field rutting
- Correlation factors must be validated locally
- Accelerated pavement testing (APT) preferred over lab-only claims

### 5. Distress Measurement Method Not Standardized
- IRI: profilometer type and calibration must be stated
- Rutting: measurement device (3-point, multi-point, laser) affects results
- PCI: ASTM D6433 procedure must be followed

### 6. Overlay Design Not Justified
- Overlay thickness must be based on deflection analysis or mechanistic method
- Do not use empirical rules without local calibration
- Bond condition between layers must be addressed

### 7. LCCA Incomplete
- Must include agency cost + user cost + salvage value
- Discount rate and analysis period must be stated
- Sensitivity analysis on key cost parameters required

### 8. Comparison Fairness
- Compare rehabilitation strategies at same baseline condition
- Include a "do nothing" or minimum treatment scenario
- Report all costs in same year dollars

### 9. Statistical Rigor
- Multiple test sections required (not just one per strategy)
- Variability reported (COV for material properties, SD for performance)
- Confidence intervals for performance predictions

### 10. Missing Practical Guidance
- Recommended treatment timing with trigger values
- Applicability limits (road class, traffic range, climate)
- Implementation considerations (lane closure, user delay)

## Red Flags

- Rutting depth measured only at one time point
- IRI reported without profilometer specifications
- Performance prediction without validation data
- Overlay design without deflection data
- LCCA without user cost consideration
