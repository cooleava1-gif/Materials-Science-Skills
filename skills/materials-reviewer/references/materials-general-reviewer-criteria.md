# Universal Cross-Domain Materials Review Criteria

## Experimental design

Critical checkpoints:

1. Research question or hypothesis must be clearly stated; experimental design must directly address it.
2. Independent variables must be systematically varied with at least 3-5 levels to establish trends.
3. Control groups or reference materials must be included (blank, neat matrix, commercial benchmark).
4. Confounding variables must be identified and controlled (humidity, temperature, operator, batch).
5. Sample size and replication strategy must be justified; minimum 3 independent replicates per condition.

Common weaknesses:

- Only 2 levels tested per variable; trend cannot be established.
- Control or reference material missing; improvement not benchmarked.
- Multiple variables changed simultaneously; individual contributions cannot be separated.
- Sample preparation variability not controlled or reported.

## Data quality and statistics

Critical checkpoints:

1. Raw data or individual data points must be presented (scatter plot, box plot) alongside summary statistics; bar charts with only error bars are insufficient.
2. Error bars must indicate standard deviation (SD), standard error (SEM), or confidence interval (CI); the type must be stated.
3. Statistical significance testing (t-test, ANOVA, regression) must be performed when differences between conditions are claimed.
4. Outlier treatment must be described (Grubbs test, IQR method); arbitrary exclusion is not acceptable.
5. Measurement uncertainty or instrument precision must be reported for key measurements.

Common weaknesses:

- Only mean values plotted without error bars or individual data points.
- Error bars present but type (SD vs. SEM) not specified; SEM used to obscure variability.
- Statistical tests not performed despite claims of "significant improvement."
- Outliers removed without justification or statistical test.

## Claims and evidence alignment

Critical checkpoints:

1. Every claim in the abstract, conclusions, and discussion must be supported by specific data or citation.
2. Magnitude of improvement must be quantified (percentage, fold-change) with statistical support.
3. Mechanism claims must be supported by direct evidence (spectroscopy, microscopy, modeling), not inferred from property changes alone.
4. Limitations of the study must be acknowledged; boundary conditions of conclusions must be stated.
5. Novelty claims must be contextualized against the current state of the art with specific comparisons.

Common weaknesses:

- "Significantly improved" claimed without quantification or statistical test.
- Mechanism proposed based on correlation, not causation; alternative explanations not considered.
- Abstract overclaims relative to the data presented.
- Limitations section absent or perfunctory.

## Reproducibility

Critical checkpoints:

1. Experimental procedures must contain sufficient detail for independent reproduction (equipment model, supplier, catalog number, exact conditions).
2. Standard test methods must be cited by full designation (ASTM, ISO, GB/T) with year.
3. Sample dimensions, conditioning, and testing environment (temperature, humidity) must be reported.
4. Data availability: raw data must be deposited in a repository or available as supplementary material.
5. Custom code, scripts, or analysis routines must be shared or described in sufficient detail.

Common weaknesses:

- Procedure described as "similar to [reference]" without listing actual conditions.
- Standard test method cited but specific edition/year not given.
- Testing environment (temperature, humidity) not recorded.
- Raw data not available; only processed results presented.

## Manuscript quality

Critical checkpoints:

1. Figures must be legible at publication size with labeled axes, units, and sufficient font size (minimum 8 pt after reduction).
2. Tables must include units, uncertainty values, and sample size for each reported value.
3. References must be current and relevant; seminal works and recent advances (last 5 years) must be cited.
4. Nomenclature must be consistent throughout the manuscript; abbreviations must be defined at first use.
5. Supplementary information must contain essential data, not be used to hide unfavorable results.

Common weaknesses:

- Figures with illegible text, missing units, or inconsistent formatting.
- Key data relegated to supplementary information while favorable data are in main text.
- Reference list outdated; missing recent high-impact publications in the field.
- Inconsistent terminology or undefined abbreviations.
