# Bridge Engineering Reviewer Criteria

## Top 10 Reviewer Concerns for Bridge / Structural Papers

### 1. Bridge Description Incomplete
- Must include: type, span length, width, age, material grades, traffic level
- Structural drawings or cross-sections required
- Loading history (posted restrictions, rehabilitation history)

### 2. Load Test Protocol Not Standardized
- Reference AASHTO Manual for Bridge Evaluation or equivalent
- Load truck weight, axle spacing, and transverse position must be reported
- Multiple load cases required (not just one truck position)

### 3. Sensor Layout Not Justified
- Sensor locations must correspond to critical structural sections
- Temperature compensation required for strain gauges
- Sampling rate and duration must match expected frequencies

### 4. FEM Not Validated
- Model must be calibrated with at least measured deflection AND frequency
- Material properties must be from testing, not assumed
- Boundary conditions must reflect actual support conditions

### 5. Load Rating Methodology Not Clear
- State code (AASHTO LRFR, Eurocode, national standard)
- Distinguish inventory vs operating rating
- Dynamic load allowance (IM) must be applied correctly

### 6. Deterioration Model Not Calibrated
- Model must be fitted to field condition data, not just lab data
- Minimum 2 data points per condition state for regression
- Exposure conditions (marine, de-icing, urban) must be stated

### 7. Corrosion Claims Without Electrochemical Data
- Half-cell potential or linear polarization required
- Do not claim corrosion from visual inspection alone
- Corrosion rate (mpy or µA/cm²) must be quantified

### 8. Strengthening Effectiveness Not Proven
- Before-and-after load test required
- Failure mode must be ductile, not brittle debonding
- Long-term durability of strengthening system addressed

### 9. Statistical Basis Missing
- Minimum 3 load test runs for repeatability
- Report COV for measured deflections and strains
- Confidence interval for rating factors

### 10. Missing Practical Guidance
- Recommended posting limits if rating is deficient
- Inspection interval recommendations
- Cost estimate for recommended rehabilitation

## Red Flags

- Rating factor reported without specifying code edition
- FEM validated with only one measurement point
- Deterioration model extrapolated beyond 2× training data range
- SHM system described but no field data presented
- Strengthening design without connection detail
