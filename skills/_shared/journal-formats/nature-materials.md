# Nature Materials — Formatting Facts

Authoritative facts about Nature Materials formatting requirements. This file holds the **facts**; each skill's `static/fragments/journal/nature-materials.md` adds the **action layer**.

> Always verify against the journal's current Guide for Authors before final submission. The numbers below are accurate as of early 2026.
> Official page: https://www.nature.com/nmat/for-authors

## Article types and limits

| Article type | Word limit | Abstract | References | Figures/Tables | Methods |
|---|---|---|---|---|---|
| **Article** | No strict word limit (typically 3,000–5,000) | Short, ~150 words, unstructured | ~50–80 typical | Flexible; Methods section separate | End of main text |
| **Letter (Brief Communication)** | ~2,000 | Short, ~100 words | ~30 | ~4 | End of main text |
| **Review** | Longer (typically 6,000–10,000) | Unstructured (~200 words) | Comprehensive (100+) | Flexible | N/A |

## Abstract

- Unstructured, concise prose
- ~150 words maximum for Articles
- No citations in abstract
- Must convey the key finding and its significance clearly
- Should be accessible to a broad scientific audience

## Figures and tables

| Parameter | Requirement |
|---|---|
| Resolution (halftone/photo) | 300 dpi minimum |
| Resolution (line art) | 1,000 dpi minimum |
| Resolution (combination) | 600 dpi minimum |
| File formats | TIFF, EPS, PDF, PNG |
| Color mode | Color online free; color in print may incur charges |
| In-figure font | ≥6 pt at final size |
| Panel labels | Lowercase bold letters: **a**, **b**, **c** |

## References

- **Style**: Numbered, sequential by first appearance (Nature style)
- **In-text**: Superscript numbers¹ or bracketed [1]
- **Format example**:
  ```
  1. Smith, A. B. & Johnson, C. D. Title of article. Nature Mater. 20, 123–130 (2025).
  ```
- Journal names abbreviated per Nature convention (e.g., Nature Mater., Nat. Commun.)
- Include DOI; list all authors (no "et al." in reference list)

## Mandatory statements

- **Data Availability statement** — required
- **Code Availability statement** — required if custom code used
- **Author Contributions** — required (individual contributions)
- **Competing Interests** — required
- **Acknowledgments** — optional
- **Ethics statement** — if applicable

## Cover letter

Nature Materials editors use the cover letter for initial triage. Emphasize:

1. The key advance in one sentence
2. Why it matters beyond the immediate subfield (cross-disciplinary impact)
3. Why now — what makes this timely

## Common desk-rejection patterns

1. **Incremental advance** — improvement on known systems without conceptual breakthrough
2. **Narrow scope** — relevance limited to a single subcommunity
3. **Lack of mechanistic insight** — phenomenological observation without understanding
4. **Insufficient novelty** — extension of prior work without transformative element
5. **Poor presentation for broad audience** — too specialized in language or framing
