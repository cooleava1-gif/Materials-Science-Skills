# Acta Materialia — Formatting Facts

Authoritative facts about Acta Materialia formatting requirements. This file holds the **facts**; each skill's `static/fragments/journal/acta-materialia.md` adds the **action layer**.

> Always verify against the journal's current Guide for Authors before final submission. The numbers below are accurate as of early 2026.
> Official page: https://www.sciencedirect.com/journal/acta-materialia/publish/guide-for-authors

## Article types and limits

| Article type | Word limit | Abstract | References | Figures/Tables | Methods |
|---|---|---|---|---|---|
| **Full paper** | No strict word limit (typically 5,000–8,000) | Structured or unstructured (~250 words) | ~40–60 typical | Flexible | Within main text |
| **Fast Track** | ~3,000 | ~150 words | ~25 | ~4–5 | Within main text |
| **Review** | Longer (8,000–15,000) | Unstructured (~300 words) | Comprehensive (100+) | Flexible | N/A |

## Abstract

- Structured or unstructured format accepted
- ~250 words for full papers; ~150 for Fast Track
- No citations in abstract
- Must clearly state the processing-structure-property relationship
- Include quantitative results where possible

## Figures and tables

| Parameter | Requirement |
|---|---|
| Resolution (halftone/photo) | 300 dpi minimum |
| Resolution (line art) | 1,000 dpi minimum |
| Resolution (combination) | 600 dpi minimum |
| File formats | TIFF, EPS, PDF, high-quality JPEG |
| Color mode | Color online; grayscale or color in print |
| In-figure font | Legible at publication size |
| Panel labels | Lowercase letters in parentheses: (a), (b), (c) |

## References

- **Style**: Numbered, sequential by first appearance (Elsevier numbered)
- **In-text**: Square brackets [1], [2], [1,2], [1–5]
- **Format example**:
  ```
  [1] A.B. Smith, C.D. Johnson, Acta Mater. 250 (2025) 119500.
  ```
- Journal names abbreviated per ISO 4
- Include DOI when available

## Mandatory statements

- **Data Availability statement** — required (Elsevier policy)
- **CRediT author statement** — required (individual author contributions)
- **Declaration of competing interest** — required
- **Acknowledgments** — optional but common
- **Ethics statement** — if human/animal subjects involved

## Cover letter

Acta Materialia editors focus on processing-structure-property relationships. Emphasize:

1. The materials science finding (one sentence)
2. The processing-structure-property connection (one sentence)
3. Why it matters to the structural/functional materials community

## Common desk-rejection patterns

1. **No clear materials science** — pure engineering or simulation without materials insight
2. **Single-property study** — only one property without structure or processing context
3. **Missing mechanism** — property changes without microstructural or theoretical explanation
4. **Incremental alloy design** — minor composition variation without new understanding
5. **Narrow scope** — relevant only to a very specific system without broader impact

## Open access

- Elsevier journal; hybrid open access model
- OA APC applies for open access (verify current rate)
- Default: subscription access
- Fast Track option for rapid publication
