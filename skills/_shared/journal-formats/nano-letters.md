# Nano Letters — Formatting Facts

Authoritative facts about Nano Letters formatting requirements. This file holds the **facts**; each skill's `static/fragments/journal/nano-letters.md` adds the **action layer**.

> Always verify against the journal's current Guide for Authors before final submission. The numbers below are accurate as of early 2026.
> Official page: https://pubs.acs.org/page/nalefd/submission/authors.html

## Article types and limits

| Article type | Word limit | Abstract | References | Figures/Tables | Methods |
|---|---|---|---|---|---|
| **Letter** | ~3,000 | ~150 words, unstructured | ~25–30 | ~4–5 | End of main text |
| **Letter with Review** | ~3,000 + review response | ~150 words | ~25–30 | ~4–5 | End of main text |

## Abstract

- Unstructured, concise prose
- ~150 words maximum
- No citations in abstract
- Must convey the fundamental nanoscience discovery clearly
- Emphasize the key finding and its significance in 2–3 sentences

## Figures and tables

| Parameter | Requirement |
|---|---|
| Resolution (halftone/photo) | 300 dpi minimum |
| Resolution (line art) | 1,000 dpi minimum |
| Resolution (combination) | 600 dpi minimum |
| File formats | TIFF, EPS, PDF |
| Color mode | Color online free; ACS charges for print color |
| In-figure font | ≥6 pt at final size |
| Panel labels | Lowercase letters in parentheses: (a), (b), (c) |

## References

- **Style**: ACS style (numbered)
- **In-text**: Square brackets [1], [2], [1,2], [1–5]
- **Format example**:
  ```
  [1] Smith, A. B.; Johnson, C. D. Nano Lett. 2025, 25, 1234–1240.
  ```
- Journal names: ACS abbreviated format
- Include DOI; list all authors

## Mandatory statements

- **Data Availability statement** — required (ACS policy)
- **Author Contributions** — required
- **Conflict of Interest** — required (ACS disclosure form)
- **Acknowledgments** — optional
- **Supporting Information** — strongly encouraged for Letters

## Cover letter

Nano Letters prioritizes rapid communication of fundamental discoveries. Emphasize:

1. The fundamental nanoscience finding (one sentence)
2. Why this is a significant advance in the field
3. Timeliness — why rapid publication matters

## Common desk-rejection patterns

1. **Not fundamental** — applied work without nanoscale mechanistic insight
2. **Incremental** — minor variation on known nanostructures or properties
3. **Too applied** — engineering application without fundamental discovery
4. **Insufficient novelty** — similar to recently published nanoscience work
5. **Weak characterization** — nanostructure not adequately verified

## Open access

- ACS journal; hybrid open access model
- OA APC applies (verify current rate, ACS AuthorChoice)
- Default: subscription access
- Fast-track review available for urgent findings
