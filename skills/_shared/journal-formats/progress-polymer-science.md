# Progress in Polymer Science — Formatting Facts

Authoritative facts about Progress in Polymer Science formatting requirements. This file holds the **facts**; each skill's `static/fragments/journal/progress-polymer-science.md` adds the **action layer**.

> Always verify against the journal's current Guide for Authors before final submission. The numbers below are accurate as of early 2026.
> Official page: https://www.sciencedirect.com/journal/progress-in-polymer-science/publish/guide-for-authors

## Article types and limits

| Article type | Word limit | Abstract | References | Figures/Tables | Methods |
|---|---|---|---|---|---|
| **Review article** | No strict limit (typically 8,000–20,000) | Unstructured (~300 words) | Comprehensive (150–300+) | Flexible | N/A |
| **Mini-review** | ~6,000–8,000 | Unstructured (~200 words) | ~100–150 | Flexible | N/A |

## Abstract

- Unstructured, comprehensive prose
- ~300 words for full reviews; ~200 for mini-reviews
- No citations in abstract
- Must provide a clear overview of the review scope and key conclusions
- Should be informative enough to stand alone

## Figures and tables

| Parameter | Requirement |
|---|---|
| Resolution (halftone/photo) | 300 dpi minimum |
| Resolution (line art) | 1,000 dpi minimum |
| Resolution (combination) | 600 dpi minimum |
| File formats | TIFF, EPS, PDF, high-quality JPEG |
| Color mode | Color online; grayscale or color in print |
| In-figure font | Legible at publication size |
| Panel labels | Lowercase letters in parentheses: (a), (b), (c) |

## References

- **Style**: Numbered, sequential by first appearance (Elsevier numbered)
- **In-text**: Square brackets [1], [2], [1,2], [1–5]
- **Format example**:
  ```
  [1] A.B. Smith, C.D. Johnson, Title of article, Prog. Polym. Sci. 150 (2025) 101234.
  ```
- Journal names abbreviated per ISO 4
- Include DOI when available

## Mandatory statements

- **Data Availability statement** — required (Elsevier policy)
- **CRediT author statement** — required (individual author contributions)
- **Declaration of competing interest** — required
- **Acknowledgments** — optional but common

## Cover letter

Progress in Polymer Science publishes invited and pre-approved reviews. If proposing:

1. Provide a detailed outline with section structure
2. Demonstrate comprehensive expertise in the topic
3. Show that no recent review covers the same scope
4. Include estimated timeline for completion

## Common desk-rejection patterns

1. **Unsolicited submission** — reviews require invitation or editor pre-approval
2. **Overlap with recent reviews** — topic already covered comprehensively
3. **Insufficient breadth** — too narrow for a review journal
4. **Lack of critical analysis** — mere summary without synthesis or perspective
5. **Outdated coverage** — missing recent key developments in a fast-moving field

## Open access

- Elsevier journal; hybrid open access model
- OA APC applies for open access (verify current rate)
- Default: subscription access
- Invited reviews may have different arrangements
