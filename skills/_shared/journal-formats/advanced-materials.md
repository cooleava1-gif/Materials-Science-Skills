# Advanced Materials — Formatting Facts

Authoritative facts about Advanced Materials formatting requirements. This file holds the **facts**; each skill's `static/fragments/journal/advanced-materials.md` adds the **action layer**.

> Always verify against the journal's current Guide for Authors before final submission. The numbers below are accurate as of early 2026.
> Official page: https://onlinelibrary.wiley.com/page/journal/15214095/homepage/forauthors.html

## Article types and limits

| Article type | Word limit | Abstract | References | Figures/Tables | Methods |
|---|---|---|---|---|---|
| **Full Paper** | ~6,000–8,000 | ~200 words, unstructured | ~40–60 typical | Flexible | End of main text or SI |
| **Communication** | ~4,000 | ~150 words | ~25–30 | ~5–6 | End of main text |
| **Review** | Longer (8,000–15,000) | ~200 words | Comprehensive (100+) | Flexible | N/A |

## Abstract

- Unstructured, concise prose
- ~200 words for Full Papers; ~150 for Communications
- No citations in abstract
- Graphical abstract required for all article types
- Must highlight novelty and significance clearly

## Figures and tables

| Parameter | Requirement |
|---|---|
| Resolution (halftone/photo) | 300 dpi minimum |
| Resolution (line art) | 1,000 dpi minimum |
| Resolution (combination) | 600 dpi minimum |
| File formats | TIFF, EPS, PDF, high-quality JPEG |
| Color mode | Color online free; check print charges |
| In-figure font | Legible at publication size |
| Panel labels | Lowercase letters: (a), (b), (c) |
| Graphical abstract | Required (Wiley standard dimensions) |

## References

- **Style**: Numbered, sequential by first appearance (Wiley numbered style)
- **In-text**: Square brackets [1], [2], [1,2], [1–5]
- **Format example**:
  ```
  [1] A. B. Smith, C. D. Johnson, Adv. Mater. 2025, 37, 2401234.
  ```
- Journal names abbreviated per standard convention
- Include DOI when available

## Mandatory statements

- **Data Availability statement** — required
- **Author Contributions** — required (CRediT taxonomy)
- **Conflict of Interest** — required
- **Acknowledgments** — optional but common
- **Supporting Information** — encouraged for supplementary data

## Cover letter

Editors use the cover letter for triage. State explicitly:

1. What is the key novelty (one sentence)
2. Why it matters to the materials science community (one sentence)
3. Cross-disciplinary relevance or application potential

## Common desk-rejection patterns

1. **Incremental work** — minor improvement on known systems
2. **Lacks novelty** — similar to previously published work
3. **Narrow impact** — relevant only to a very specific subfield
4. **Weak mechanistic understanding** — phenomenological without insight
5. **Poor graphical abstract** — fails to convey the key message visually
