# British English for Materials Manuscripts

Most UK and European materials journals (Taylor & Francis, some Springer,
some Wiley) require British English. Elsevier journals accept both but prefer
consistency. This file defines the key spelling, vocabulary, and grammar
differences.

---

## Spelling Differences

### -our / -or

| American | British |
|---|---|
| behavior | behaviour |
| color | colour |
| favor | favour |
| honor | honour |
| labor | labour |
| neighbor | neighbourhood |
| vapor | vapour |

### -ize / -ise

| American | British | Notes |
|---|---|---|
| analyze | analyse | Both acceptable; be consistent |
| characterize | characterise | |
| optimize | optimise | |
| standardize | standardise | |
| summarize | summarise | |
| recognize | recognise | |

**Exception**: `size`, `prize`, `seize` remain the same in both.

### -yze / -yse

| American | British |
|---|---|
| analyze | analyse |
| catalyze | catalyse |
| paralyze | paralyse |

### -er / -re

| American | British |
|---|---|
| center | centre |
| fiber | fibre |
| meter | metre |
| caliber | calibre |

**Exception**: `thermometer`, `diameter`, `parameter` — same in both.

### -ed / -t

| American | British |
|---|---|
| learned | learnt |
| burned | burnt |
| spelled | spelt |

**Note**: Both forms are acceptable in British English; `-t` is more traditional.

## Grammar Differences

### Collective nouns

British English treats collective nouns as plural when emphasizing individuals:

| American | British |
|---|---|
| `The team was...` | `The team were...` |
| `The group has...` | `The group have...` |

In materials writing, prefer singular for clarity:
> `The research group was...` (both acceptable)

### Prepositions

| American | British |
|---|---|
| `on the weekend` | `at the weekend` |
| `through Friday` | `until Friday` |
| `different from` / `different than` | `different from` / `different to` |

In academic writing, `different from` is preferred in both varieties.

### Dates

| American | British |
|---|---|
| `June 14, 2026` | `14 June 2026` |

Check journal style guide; most materials journals accept either.

## Consistency Rules

### Rule 1: Pick one variety and stick to it

Do not mix American and British spelling in the same manuscript. Choose one
and use it throughout.

### Rule 2: Check the journal

| Journal family | Preferred English |
|---|---|
| Elsevier (CBM, CCC, Fuel, JCP) | Either; consistency required |
| Taylor & Francis (RMPD, IJPE) | British preferred |
| ASCE (JMCE) | American |
| Springer (JBE) | Either; check specific journal |
| Wiley (JACERS) | Either; check specific journal |

### Rule 3: Compound modifiers

Use hyphens for compound modifiers before nouns:

| Correct | Notes |
|---|---|
| `water-based coating` | Compound modifier before noun |
| `high-temperature test` | Compound modifier before noun |
| `the coating is water based` | No hyphen after noun |
| `the test was at high temperature` | No hyphen after noun |

### Rule 4: -ing forms as nouns

| Correct | Notes |
|---|---|
| `The curing of the epoxy...` | Gerund as noun |
| `Curing the epoxy at 60°C...` | Gerund as verb |
| `The curing process...` | Gerund as adjective (no hyphen needed) |

## Quick Reference: Common Corrections

| Wrong (mixed/inconsistent) | Correct (British) |
|---|---|
| `The material's behavior and colour` | Pick one: `behavior` or `behaviour` |
| `The center of the fibre` | `The centre of the fibre` |
| `Modeling the modelling` | `Modelling` throughout |
| `Analyse the analysis` | Fine (verb/noun forms differ) |
| `The program was programmed` | `The programme was programmed` (general) / `The program was programmed` (computing) |

---

## Audit Checklist

- [ ] All spelling is consistent (all American or all British).
- [ ] No mixed `-ize`/`-ise` in the same manuscript.
- [ ] No mixed `-or`/`-our` in the same manuscript.
- [ ] No mixed `-er`/`-re` in the same manuscript.
- [ ] Double consonants are consistent (`modelling` not `modeling` in British).
- [ ] Compound modifiers are hyphenated before nouns.
- [ ] Journal-specific requirements are met.
