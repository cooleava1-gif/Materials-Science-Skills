# Nanomaterials — Polishing Guide

## Terminology to verify

| Term | Domain-specific check |
|---|---|
| particle size | TEM primary size or DLS hydrodynamic? Method must be stated |
| monodisperse | Requires size distribution (PDI < 0.1 for DLS; narrow TEM histogram) |
| quantum confinement | Requires size-dependent optical property shift (UV-vis/PL) |
| surface functionalization | Requires at least two of: FTIR, XPS, zeta potential |
| colloidal stability | Must specify: medium, concentration, time, monitoring method |

## Common overclaim patterns

| Overclaim | Safer alternative |
|---|---|
| "Uniform nanoparticles were synthesized" | "TEM shows [morphology] particles with mean size [value] ± [error] nm (n = [number])" |
| "The nanomaterial significantly enhanced performance" | "[Property] increased by [X]% at [loading] wt% compared to [control]" |
| "Green synthesis" | Requires comparison of energy, solvent, or byproduct vs conventional method |
| "The material is non-toxic" | Requires specific cell viability assay, concentration range, and exposure time |

## Hedging guide

| Evidence level | Verb | Example |
|---|---|---|
| TEM of ≥100 particles + statistics | demonstrates, yields | "TEM demonstrates a mean particle size of [value] nm with σ = [error]" |
| DLS + TEM (no statistics) | indicates, suggests | "DLS indicates a hydrodynamic size of [value] nm, consistent with TEM" |
| Single characterization | appears, may suggest | "XRD peak broadening suggests a crystallite size in the nanometer range" |

## Journal-specific language

- Report both TEM primary size AND DLS hydrodynamic size; explain the difference.
- "Monodisperse" requires evidence: either PDI < 0.1 (DLS) or narrow TEM histogram with ≥100 counts.
- For 2D materials: specify layer number determination method (AFM, Raman, TEM).
- For catalysis: normalize activity to surface area (BET), not just mass.
