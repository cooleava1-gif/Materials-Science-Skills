# Reader Note — Domain Context

Loaded automatically when the domain is metals/alloys.

When reading a metals paper, pay special attention to:
- Full composition (wt%) and alloy designation.
- Complete processing history (casting, deformation, heat treatment, cooling rate).
- Microstructural evidence: grain size, phase identification, precipitate analysis.
- Both strength and ductility/toughness data.
- Specimen orientation and test standards.

Record the composition, processing parameters, microstructural features, and mechanical properties along with test standards.
