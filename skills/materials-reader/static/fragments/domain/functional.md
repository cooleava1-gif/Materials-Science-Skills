# Reader Note — Domain Context

Loaded automatically when the domain is functional/electronic materials.

When reading a functional materials paper, pay special attention to:
- Synthesis/deposition method and conditions.
- Structural characterization confirming phase and quality (XRD, Raman, XPS).
- Measurement conditions (temperature, illumination, frequency, scan rate).
- Device architecture and active area.
- Performance metrics with error margins and number of devices tested.

Record the material system, synthesis details, key functional properties, and device performance with measurement context.
