# Metals Domain Data Rules

For metals and alloys data, track:

- Alloy grade, composition (major + trace elements), standard designation.
- Processing route and full parameter set (casting, rolling, forging, heat treatment, welding).
- Heat treatment schedule (austenitizing temp/time, quench medium, tempering temp/time).
- Microstructure characterization method and measured features (grain size, phase fraction, precipitate).
- Test standard, specimen orientation (LT, TL, ST), strain rate, temperature.
