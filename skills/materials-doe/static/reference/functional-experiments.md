# Functional Materials Experiment Design Reference

## Typical factors
- Composition: doping level, stoichiometry, solid solution ratio.
- Synthesis: temperature, time, atmosphere, precursor concentration.
- Processing: deposition parameters, annealing conditions, quenching rate.
- Device architecture: layer thickness, electrode material, active area.

## Typical responses
- Structural: phase purity, crystallinity, lattice parameter, defect density.
- Optical: bandgap, transmittance, PL intensity, refractive index.
- Electrical: conductivity, mobility, carrier concentration, dielectric properties.
- Device: efficiency, EQE, responsivity, stability (retention % vs time).

## Recommended design
- Full factorial for composition optimization.
- RSM for synthesis/processing optimization.
- Custom design for multi-layer device optimization.
