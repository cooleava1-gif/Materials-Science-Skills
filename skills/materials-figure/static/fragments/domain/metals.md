# Metals and Alloys — Figure Guide

## Common figure types

| Figure type | When to use | Key requirements |
|---|---|---|
| Stress-strain curve | Tensile properties | Report UTS, yield, elongation; multiple curves per condition |
| Optical micrograph (OM) | Grain structure, phase distribution | Etching method; magnification; scale bar; grain size annotation |
| SEM fractography | Fracture surface analysis | Two magnifications (low overview + high detail); failure mode labels |
| EBSD map | Grain orientation, texture, phase ID | Color key for orientation; IPF legend; step size reported |
| TEM micrograph | Precipitate, dislocation, nano-features | Scale bar; precipitate size distribution if relevant |
| XRD pattern overlay | Phase identification, texture | Peak labels; ICDD reference pattern; 2θ range |
| Hardness profile | Across-weld, through-thickness | Indentation locations; load and dwell time |

## Typical multi-panel structure

**Panel A**: Microstructure (OM/EBSD showing grain structure)
**Panel B**: Mechanical (stress-strain curves or bar charts)
**Panel C**: Fractography (SEM of fracture surface)
**Panel D**: Property comparison (strength-ductility map or bar chart)

## Reviewer-sensitive pitfalls

- ⚠️ Stress-strain curves without reporting specimen orientation.
- ⚠️ Fracture mechanism claimed without fractography.
- ⚠️ EBSD maps without step size or indexing rate.
- ⚠️ Grain size claimed without measurement method (linear intercept, ASTM E112).
- ⚠️ Hardness values without load/indenter specification.

## Caption boundary phrases

- `Engineering stress-strain curves of [alloy] after [heat treatment]. Specimens oriented [direction].`
- `SEM fractographs of tensile specimens: (a) [condition A] showing [feature], (b) [condition B].`
- `EBSD IPF maps of [alloy]: (a) as-received, (b) after [processing]. Step size: [value] µm.`
- `XRD patterns of [alloy] identifying [phases]. Reference patterns from ICDD [numbers].`
