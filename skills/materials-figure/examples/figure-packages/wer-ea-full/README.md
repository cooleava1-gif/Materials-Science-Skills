# WER-EA Full Figure Package

Golden package for a WER-EA manuscript or mini-review figure. It combines a
mechanism panel, performance panel, evidence heatmap, and literature screening
flow into a 2x2 narrative.

Panels:

- A mechanism map: demulsification, curing, interfacial bonding, and emulsion stability.
- B performance window: bonding strength and workable dosage range.
- C evidence heatmap: characterization and durability evidence coverage.
- D literature screening: source flow from search to included evidence rows.

Use this package as a full figure package template, not as manuscript evidence
until real source data replace the synthetic rows.
