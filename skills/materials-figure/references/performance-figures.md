# Performance Figures

> **Domain context**: The `domain` axis has loaded domain-specific figure guidance. The list below is universal; supplement with domain-specific figure types from the domain guide.

Use for strength, bonding, viscosity, modulus, efficiency, or other property data.

Recommended figure types:

- grouped bar for control versus modified groups,
- line plot for dosage-response (include full range, not just the rising branch),
- trade-off plot for performance versus competing requirement,
- retention ratio plot for conditioned/unconditioned data,
- scatter plot with regression for structure-property correlation.

Always show units and test conditions. Report error bars (SD or CI) for all quantitative comparisons.

## Domain-specific emphases

- **Civil**: dosage-response curves must extend past the optimum to show the descending branch.
- **Polymers**: report both strength and elongation on the same plot to show the trade-off.
- **Metals**: stress-strain curves preferred over bar charts; show full curves.
- **Ceramics**: include Weibull statistics for brittle fracture data.
- **Functional**: scatter or bar with individual data points to show device-to-device variation.
- **Nanomaterials**: size-dependent property plots (property vs particle size).
