# Nano Figure Package

Recommended package for nanomaterial characterization:

1. Synthesis route schematic and particle size distribution (DLS histogram or TEM size histogram),
2. TEM/SEM morphology plate showing size, shape, and dispersion state,
3. XRD pattern with phase identification and crystallite size (Scherrer),
4. UV-Vis absorption or photoluminescence spectra showing optical properties,
5. TGA/DSC thermal stability and grafting density evidence,
6. Property enhancement: mechanical, thermal, or functional property vs filler loading (with percolation threshold if applicable),
7. FTIR or XPS surface chemistry evidence for functionalization.

Caption stance: size-distribution width and dispersion quality are more important than "smaller is better". Report measurement method (TEM vs DLS) alongside particle size claims.
