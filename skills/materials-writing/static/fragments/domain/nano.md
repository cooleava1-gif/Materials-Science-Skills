# Nanomaterials — Writing Guide

Loaded automatically when the domain is nanomaterials (nanoparticles, 2D materials, thin films, nanocomposites).

## Domain narrative arc

The strongest arc for nanomaterials follows a **synthesis-structure-property-size-effect** logic:

1. Application context: what unique property of nanomaterials enables a new or improved function.
2. Synthesis challenge: controlling size, shape, distribution, and surface chemistry.
3. Design strategy: synthesis method selection, precursor, capping agent, template, or post-synthesis treatment.
4. Structural evidence: TEM, SEM, AFM, XRD, Raman, UV-vis, DLS confirming size, shape, and distribution.
5. Property evidence: the size/structure-dependent property (optical, electrical, catalytic, mechanical, magnetic).
6. Application demonstration: how the nanomaterial performs in the target context.
7. Boundary: aggregation, long-term stability, toxicity, scalability of synthesis.

## Key evidence chain

Synthesis → size/shape/morphology (TEM, SEM, DLS) → structure (XRD, Raman, XPS) → property → application performance

Size distribution statistics (minimum 100 particles from TEM images) are required for any nanoparticle study.

## Section guidance

### Introduction
- Emphasize the **size/structure-dependent property** that motivates the work.
- Gap: what size range, morphology, surface chemistry, or dispersion state has not been achieved.
- Keep background brief; focus on the unresolved synthesis-property link.

### Methods
- Synthesis: precursor concentration, temperature, time, surfactant/capping agent, purification method.
- Characterization: instrument details, sample preparation method (especially for TEM).
- For 2D materials: exfoliation method, layer number determination, substrate, transfer process.
- For nanocomposites: dispersion method, loading fraction, interfacial treatment.

### Results
- Morphology first: representative TEM/SEM images with size distribution histogram.
- Structure confirmation: XRD pattern with crystallite size (Scherrer), Raman spectrum, UV-vis.
- Property: correlated with size/morphology.
- For nanocomposites: show dispersion quality.

### Discussion
- Explain how size, shape, surface chemistry, or dispersion influences the measured property.
- Compare with literature values for similar nanomaterials.
- Address aggregation and stability—the most common limitation.

### Conclusion
- State the synthesis method, achieved size/morphology, and the resulting property.
- Include limitations: batch-to-batch reproducibility, colloidal stability, scale-up feasibility.
- Avoid "promising for [application]" without specifying the performance gap.

## Reviewer-sensitive points in nanomaterials

- ⚠️ Particle size claimed from TEM without counting statistics or from XRD alone.
- ⚠️ "Monodisperse" claimed without showing size distribution.
- ⚠️ Aggregation not addressed in dispersion or composite studies.
- ⚠️ Surface chemistry claims without XPS, FTIR, or zeta potential evidence.
- ⚠️ Toxicity or biocompatibility claimed without relevant biological testing.

## Common pitfalls

| Pitfall | Example | Fix |
|---|---|---|
| Size from XRD only | crystallite size reported as particle size | Use TEM for primary particle size |
| Missing statistics | "average size 10 nm" from 5 particles | Count ≥100 particles |
| Confusing hydrodynamic with primary size | DLS reported as particle size without context | Report both TEM and DLS; explain difference |
| Overclaiming uniformity | "uniform size distribution" without histogram | Show size distribution plot |
| Ignoring batch variability | results from one synthesis batch | Test at least 3 independent batches |
