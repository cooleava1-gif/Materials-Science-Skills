# Nanomaterials Domain Citation Guide

For nanomaterials manuscripts, keep evidence separated by:

- **Synthesis method**: chemical, physical, biological; precursor, capping agent, template.
- **Size/morphology**: TEM size distribution, DLS hydrodynamic size, aspect ratio, shape.
- **Structural characterization**: XRD (crystallite size, phase), Raman, UV-vis, PL.
- **Surface properties**: functionalization, zeta potential, surface area (BET).
- **Size-dependent properties**: optical, catalytic, magnetic, mechanical.
- **Application context**: device configuration, loading, dispersion medium, stability.

Separate synthesis-characterization citations from property/application citations. Report size statistics (≥100 particles for TEM).
