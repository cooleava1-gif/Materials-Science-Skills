# Functional / Electronic Materials Domain Citation Guide

For functional materials manuscripts, keep evidence separated by:

- **Material class**: semiconductor, dielectric, piezoelectric, photonic, electrochemical.
- **Synthesis method**: deposition (CVD, PVD, sputtering), sol-gel, hydrothermal, solid-state.
- **Structural evidence**: XRD, Raman, XPS, TEM confirming phase, epitaxy, defects.
- **Functional properties**: bandgap, conductivity, permittivity, polarization, efficiency.
- **Device performance**: configuration, active area, measurement conditions, stability.
- **Benchmark comparison**: state-of-the-art values for the same material class.

Separate synthesis-structure citations from device performance citations.
