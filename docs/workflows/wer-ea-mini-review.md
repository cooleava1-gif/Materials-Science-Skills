# WER-EA mini-review

## Route Summary

Routes a waterborne epoxy modified emulsified asphalt topic through literature
screening, evidence mapping, mini-review structure, and figure planning.

## Demo Prompt

```text
Help me run a WER-EA mini-review workflow from screening to figure planning.
```

## Workflow Steps

1. `materials-research` resolves the WER-EA domain and journal family.
2. `materials-citation` builds the search plan and citation matrix.
3. `materials-reader` maps claims, mechanisms, figures, and source anchors.
4. `materials-writing` drafts the review arc and evidence gaps.
5. `materials-figure` plans the dosage-window, durability, and heatmap figures.

## Expected Artifacts

- Search-plan table and screened citation matrix.
- Reader package with claim-evidence-mechanism-boundary rows.
- Mini-review outline with missing-evidence flags.
- Figure contract for the WER-EA proof board route.

## What Good Looks Like

Claims remain bounded to screened sources, mechanism language stays hedged, and
the figure plan separates measured performance from inferred interface logic.
