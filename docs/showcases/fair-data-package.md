# FAIR data package

## Outcome Snapshot

A FAIR data package organizes raw and processed files, metadata, audit notes,
and a journal-ready data-availability statement.

## Demo Prompt

```text
Build a FAIR supplementary data package for this materials manuscript.
```

## Proof Assets

- `docs/gallery/gallery_material_systems.png`

## Build Path

1. `materials-data` selects the domain schema.
2. `materials-data` writes dataset README and metadata.
3. `materials-data` runs the FAIR audit checklist.
4. `materials-writing` or `materials-polishing` can adapt the availability text.

## When To Use This Route

Use it before submission when datasets, supplementary tables, or repository
statements need to be organized and checked.
